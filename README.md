# UtiliPad

UtiliPad es una aplicación web completa organizada como monorepo que incluye frontend y backend.

## 📁 Estructura del Proyecto

```
UtiliPad/
├── Front/          # Frontend - Aplicación Astro
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── ...
├── Back/           # Backend - (En desarrollo)
│   └── ...
├── README.md       # Este archivo
└── .gitignore
```

## 🚀 Inicio Rápido

### Frontend (Astro)

```bash
cd Front
pnpm install
pnpm dev
```

### Backend

*En desarrollo - instrucciones por venir*

## 📝 Desarrollo

Este proyecto utiliza una estructura de monorepo para facilitar el desarrollo conjunto del frontend y backend.

### Frontend
- **Framework**: Astro
- **Package Manager**: pnpm
- **Ubicación**: `./Front/`

### Backend
- **Framework**: Por definir
- **Ubicación**: `./Back/`
